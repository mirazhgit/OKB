# ОКБ Банк — Официальный сайт и Интернет-банк

Репозиторий фронтенд-части официального сайта и интернет-банка «ОКБ». Проект представляет собой многостраничный сайт, разработанный на чистом HTML, CSS и JavaScript (Vanilla JS), с фокусом на адаптивность и кросс-браузерность.

## 🌐 Функциональность

Проект состоит из двух основных частей:

1.  **Публичный сайт (Лендинг):**
    - Главная страница с промо-блоками и калькулятором кредитов/ипотеки.
    - Каталог банковских продуктов (дебетовые карты White и WhiteGold).
    - Информационные страницы (курсы валют, описание продуктов).
2.  **Приватный кабинет клиента:**
    - Страница входа и регистрации (с переключением способов входа: логин/телефон/карта).
    - Дашборд с информацией о счетах и картах, отображением баланса и переключением валют (рубль/доллар).

## 🛠 Стек технологий

- **HTML5:** Семантическая верстка.
- **CSS3:** Адаптивная верстка, Flexbox/Grid, кастомные стили без использования фреймворков.
- **JavaScript (Vanilla):** Чистый JS для интерактивности:
  - Переключение темы (светлая/темная).
  - Работа с выпадающими списками (кастомный селект валют).
  - Интерактивные слайдеры (range) в кредитном калькуляторе.
  - Переключение табов (категории карт).
- **jQuery (частично):** Используется на странице регистрации для маски ввода (телефон, дата, паспорт) и календаря (jQuery UI).

## 📁 Структура проекта

```
/
├── index.html               # Главная страница
├── account.html             # Личный кабинет (сводка)
├── catalog-maps.html        # Каталог дебетовых карт
├── debitcard.html           # Детальная страница карты White
├── entrance.html            # Страница входа
├── registration.html        # Страница регистрации
├── exchangerate.html        # Курсы валют
├── assets/
│   ├── style/               # CSS файлы (отдельный для каждой страницы)
│   ├── script/              # JS файлы (логика для каждой страницы)
│   ├── img/                 # Изображения и иконки
│   └── fonts/               # Шрифты
```
